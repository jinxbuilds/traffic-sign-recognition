# Traffic Sign Recognition System

CNN-based traffic sign classifier trained on the German Traffic Sign Recognition Benchmark (GTSRB).

## Features
- Dataset preprocessing
- Data augmentation
- CNN training
- Traffic sign prediction
- Batch evaluation
- OpenCV inference pipeline

## Tech Stack
- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Scikit-image

## Dataset
GTSRB Dataset:
https://benchmark.ini.rub.de/gtsrb_news.html

Dataset is not included in this repository.
